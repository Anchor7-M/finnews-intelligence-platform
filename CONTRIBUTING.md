# Contributing

This repository is currently a local portfolio build. Contributions should preserve the zero-cost, metadata-first, synthetic-demo defaults.

## Development

1. Use Python 3.12 and Node 20.
2. Install Python dependencies in `.venv`.
3. Install frontend dependencies in `frontend/node_modules`.
4. Run `python scripts/dev.py verify-lite` before proposing changes.

## Data Policy

Do not commit copied full article bodies, proprietary corpora, secrets, generated model artifacts, or paid-service credentials.
