# finnews-paper-execution-v1

This contract describes paper execution intents only. It is not a broker order,
does not contain account or credential fields, and is not investment advice.

Unknown fields are rejected. Broker, MT5 terminal, account, credential, order
ticket, lot, leverage, margin, stop-loss, take-profit, order type, execution
flag, `order_check`, `order_send`, `TRADE_ACTION`, `ORDER_TYPE`, and
`MqlTradeRequest` fields are forbidden.
