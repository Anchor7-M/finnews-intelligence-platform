# Official Data Asset Mapping

Milestone 3B maps official data series/query profiles to the existing M3A
canonical asset universe through `SeriesAssetAssociation`.

These mappings are research relevance hypotheses only:

- no direction;
- no price target;
- no recommendation;
- no executable order field;
- no MT5 terminal integration.

The synthetic fixture contains 80 associations across 10 profiles. The mapping
is intentionally not all-to-all; each profile links to a bounded subset of
assets for navigation and downstream research design.
