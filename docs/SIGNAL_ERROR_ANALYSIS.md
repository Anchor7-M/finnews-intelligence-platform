# Signal Error Analysis

M3C emits deterministic error-analysis records for synthetic scenarios and
horizons. Categories include high-confidence opposite reactions,
high-strength muted reactions, low-confidence consistent reactions,
contradictory or mixed events, stale or expired candidates, missing-bar cases,
regime-shift failures, asset-class weak spots, and event-family weak spots.

Each record stores synthetic signal ID, asset ID, event family, expected
direction, observed label, abnormal return, confidence, strength, horizon,
regime, scenario ID, and deterministic category.

The records are diagnostic buckets only. They do not provide free-form causal
claims, trading instructions, or investment advice.
